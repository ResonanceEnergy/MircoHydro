## How to Use (FreeCAD 0.20+)
1. Open **FreeCAD** → `Macro` → `Macros...` → `Execute` → select `build_all.FCMacro`.
2. The macros will create and save these files:
   - `01_Diffuser/Diffuser_vA1.FCStd`
   - `02_Nozzle/NozzleBody_9mm_vA1.FCStd`
   - `03_VanePacks/VanePack_10deg.FCStd`, `VanePack_20deg.FCStd`, `VanePack_30deg.FCStd`
   - `04_Retainer/RetainerCap_vA1.FCStd`
   - `Assembly/MicroHydroV1_Assembly.FCStd`
3. To **export STEP/STL**: open any `.FCStd` file → `File` → `Export...` → choose STEP (.step) or STL (.stl).

## Editing Parameters (Highly Parametric Option)
- Edit values in `params.json` (mm). Then re-run `build_all.FCMacro` to regenerate all parts.
- A human-friendly `MicroHydroV1_params.xlsx` mirrors the same values for documentation.

## Notes
- Units are **millimetres**.
- Geometry is generated with engineering-sensible defaults for a **9 mm nozzle** and **~88–92 mm** pack sizing.
- Assembly uses **App::Link** placements (no external workbenches required). 
- You can still use Assembly4 later if desired.

## Troubleshooting
- If the macro errors with `must be run inside FreeCAD`, ensure you launch via FreeCAD's Macro dialog.
- If parts overlap oddly, adjust spacings in `params.json` under `Assembly`.
