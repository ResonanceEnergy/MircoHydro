# DO-ALL Add-on (Templates + Automation + Closed Loop)

This add-on provides:
- Time-series templates for resonance metric
- DAQ/Logger config workbook
- Power Automate flow blueprints
- A closed-loop runner script (local)

## Locked-in resonance metric
Primary resonance metric: **T-002 PSD peakiness**
- Score = max(PSD)/median(PSD) within 0.2-20 Hz.

## Where to place files
Copy the `MicroHydroV1/` folder contents into your SharePoint SoT synced folder (merge/overwrite).
